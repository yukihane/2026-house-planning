## 気になる用語

- [x] 造成費用
- [x] 地盤改良工事
- [x] 外構
- [x] つなぎ融資
- [x] 分割融資
- [x] 団体信用生命保険 (団信)

### 🔍 用語解説

#### 1. 造成費用（ぞうせいひよう）
建物を建てるために、土地を物理的に整えるための費用です。
*   **主な内容**: 傾斜地の平坦化（切土・盛土）、崩落防止の壁（擁壁）の作成、不要な石や樹木の除去（整地）、水道・ガス管の引き込みなど。
*   **注意点**: 土地の形状や元々の用途（畑など）によって、数百万円単位で費用が変わる可能性がある「見えないコスト」の代表格です。

#### 2. 地盤改良工事（じばんかいりょうこうじ）
建物の重さに耐えられるよう、地盤の強度を人工的に補強する工事です。
*   **主な内容**: 地面の表面を固める「表層改良」、コンクリートの柱を地中に作る「柱状改良」、鉄の杭を硬い地層まで打つ「鋼管杭」など。
*   **注意点**: 地盤調査の結果が出るまで必要かどうかが分からず、必要になった場合は追加で50万〜200万円程度の予算が必要になることが多いです。

#### 3. 外構（がいこう）
建物の外側にある工作物や庭、駐車場などの総称です。「エクステリア」とも呼ばれます。
*   **主な内容**: 門、フェンス、駐車場、玄関までのアプローチ、庭、植栽、照明など。
*   **注意点**: ハウスメーカーの建物本体価格に含まれていないことが多く、別途予算（一般的に本体価格の1割程度が目安）を確保しておく必要があります。

#### 4. つなぎ融資（つなぎゆうし）
住宅ローンが実行されるまでの間、先行して必要になる資金（土地代や着工金など）を一時的に借りる短期のローンです。
*   **主な内容**: 建物完成時に住宅ローンでまとめて返済することを前提とした「橋渡し」の借入。
*   **注意点**: 住宅ローンよりも金利が高く、事務手数料や印紙代が別途かかるため、利用には一定のコストが発生します。

#### 5. 分割融資（ぶんかつゆうし）
1つの住宅ローンを、建物完成前（土地購入時や着工時）から複数回に分けて実行してもらう仕組みです。
*   **主な内容**: つなぎ融資を使わずに、住宅ローンの低金利を早い段階から適用できます。
*   **注意点**: 融資のたびに抵当権の設定費用（登録免許税など）が発生する場合があるほか、取り扱っている銀行が少ないことが挙げられます。

#### 6. 団体信用生命保険（だんたいしんようせいめいほけん／通称：団信）
住宅ローンの借主が亡くなったり、重度の障害を負ったりした場合に、保険金でローンの残高を完済する保険です。
*   **主な内容**: 家族にローンのない家を残すための仕組み。民間銀行の住宅ローンの多くで加入が義務付けられています。
*   **注意点**: 健康状態によっては加入できない（住宅ローンが組めない）場合があるため、持病がある方は早めの確認が必要です。最近では「がん保障」や「生活習慣病保障」などの特約も充実しています。
