# 主要な税制優遇と補助金制度（2026年最新版）

#### 概要
家を建てる際には、国や地方自治体による様々な税制優遇や補助金制度を利用することで、経済的な負担を大きく軽減できます。2026年現在、省エネ性能の極めて高い住宅に対する支援が強化されています。

#### 1. 住宅ローン減税（所得税・住民税の控除）
*   **制度概要**: 住宅ローンの年末残高の0.7%が最大13年間、所得税（及び一部住民税）から控除される制度。
*   **適用期間**: 2026年現在も継続中。
*   **借入限度額（2026年入居）**:
    *   **長期優良住宅・低炭素住宅**: 4,500万円（子育て・若者夫婦世帯は5,000万円）
    *   **ZEH水準省エネ住宅**: 3,500万円（子育て・若者夫婦世帯は4,500万円）
    *   **省エネ基準適合住宅**: 3,000万円（子育て・若者夫婦世帯は4,000万円）
    *   ※省エネ基準を満たさない住宅は原則対象外。
*   **主な条件**:
    *   合計所得金額2,000万円以下。
    *   床面積50㎡(約15.125坪)以上(特例で40㎡以上) など。
    *   「子育て世帯」は18歳未満の子を持つ世帯、「若者夫婦世帯」は夫婦のいずれかが39歳以下の世帯。

#### 2. 主な補助金制度

*   **未来エコ住宅2026事業（みらいエコ住宅2026）**:
    *   **概要**: 高い省エネ性能を有する新築住宅への補助金。
    *   **補助額（姫路市等）**: 
        *   **GX志向型住宅**: 110万円（全世帯対象）
        *   **長期優良住宅**: 75万円（子育て・若者世帯対象）
        *   **ZEH水準住宅**: 35万円（子育て・若者世帯対象）
    *   **詳細**: [[未来エコ住宅支援事業2026の概要|詳細はこちら]]
*   **地域型住宅グリーン化事業**:
    *   **概要**: 地域の工務店（採択グループ）で建てる高性能な木造住宅が対象。
    *   **補助額**: 最大140万円程度（加算あり）。
    *   **注意**: **大手ハウスメーカーで建てる場合は原則として対象外**となります。

#### 3. 贈与税の非課税措置
*   **制度概要**: 父母や祖父母から住宅取得資金の贈与を受けた場合、一定額まで贈与税が非課税になる特例。
*   **非課税限度額**:
    *   **質の高い住宅（省エネ・耐震・バリアフリー）**: 1,000万円
    *   **一般の住宅**: 500万円
*   **適用期限**: 2026年12月31日まで。

#### 4. 不動産関連税制の軽減措置
*   **不動産取得税**: 課税標準から最大1,300万円控除（長期優良住宅の場合）。
*   **固定資産税**: 新築から一定期間、税額を1/2に減額（長期優良住宅は5年間、一般住宅は3年間）。
*   **登録免許税**: 保存登記・移転登記の税率軽減（長期優良住宅等はさらに優遇）。

#### 5. 住宅性能による違い（用語解説）
*   **GX志向型住宅**: 断熱等級7などの極めて高い省エネ性能を有し、脱炭素社会への貢献度が高い住宅。2026年の補助金において最も手厚い支援対象。
*   **長期優良住宅**: 耐久性・耐震性・維持管理に優れた住宅。税制優遇が最もバランス良く受けられる。
*   **低炭素住宅**: 二酸化炭素排出抑制に特化した住宅。住宅ローン減税等で長期優良と同等の優遇。

### 💡 長期優良住宅 vs 低炭素住宅 の比較

| 項目 | 長期優良住宅 | 低炭素住宅 |
| :--- | :--- | :--- |
| **主な目的** | **「長く大切に住む」** | **「二酸化炭素を抑える」** |
| **主な要件** | 耐震性、耐久性、省エネ、面積、メンテナンス計画 | ZEH水準 ＋ 低炭素化措置（設備） |
| **ハードル** | 項目が多く総合的に高い。30年以上の維持保全計画が必要。 | 省エネ性能と設備に特化。維持保全計画の提出義務はない。 |
| **大手メーカー対応** | ほぼ全社が標準仕様で適合可能。 | ほぼ全社が適合可能だが、設備の微調整が必要な場合あり。 |

#### 注意点
補助金は**予算上限に達し次第終了**するため、早めの検討と事業者への確認が不可欠です。また、多くの制度で「着工前」の申請が必要となります。
