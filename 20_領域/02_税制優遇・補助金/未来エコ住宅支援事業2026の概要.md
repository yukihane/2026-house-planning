# 未来エコ住宅2026事業（みらいエコ住宅2026）の概要

## 1. 事業の概要
高い省エネ性能を有する新築住宅の取得を支援するための、2026年度（令和8年度）の国の補助金制度です。従来の「子育てエコホーム支援事業」の後継にあたります。

## 2. 補助対象世帯と補助額
お住まいの地域の「地域区分」によって補助額が異なります。（※姫路市は通常「6地域」に該当します）

| 住宅の性能基準 | 対象世帯 | 補助額（姫路市などの5〜8地域） |
| :--- | :--- | :--- |
| **GX志向型住宅** | **全世帯**（制限なし） | **110万円** |
| **長期優良住宅** | 子育て世帯・若者夫婦世帯 | **75万円** |
| **ZEH水準住宅** | 子育て世帯・若者夫婦世帯 | **35万円** |

*   **子育て世帯**: 申請時点で18歳未満の子を有する世帯。
*   **若者夫婦世帯**: 申請時点でいずれかが39歳以下の夫婦。
*   **古家除却加算**: 既存住宅を解体して新築する場合、**+20万円**が加算される場合があります（条件あり）。

## 3. 主な要件
*   **床面積**: 50㎡（約15.1坪）以上 240㎡以下であること。
*   **居住要件**: 建築主自らが居住する住宅であること。
*   **事業者登録**: 本事業に登録している住宅会社（ハウスメーカー・工務店）と契約すること。
*   **重複不可**: 国の他の補助金（給湯省エネ、窓リノベなど）と、同一の補助対象（新築本体）で併用することはできません。

## 4. スケジュール（2026年度）
予算上限に達し次第、期限前でも終了します。

*   **着工期限**: 2025年11月28日以降に基礎工事に着手したもの。
*   **申請予約の締切**: 2026年11月16日まで（ZEH水準住宅は8月17日まで）。
*   **完了報告期限**: 2027年7月31日まで。

## 5. 検討のポイント
*   **GX志向型への挑戦**: 断熱等級7などの非常に高い性能が求められますが、世帯制限がなく補助額も最大です。大手メーカー（一条工務店、積水ハウス等）の対応状況を確認する価値があります。
*   **早めの枠確保**: 住宅会社を通じて、着工前の「交付申請の予約」を確実に行ってもらうことが重要です。
