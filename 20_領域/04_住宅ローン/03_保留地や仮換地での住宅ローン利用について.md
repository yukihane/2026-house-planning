# 保留地や仮換地での住宅ローン利用について

土地区画整理事業地内の土地（保留地・仮換地）で住宅ローンを検討する際の注意点と、金融機関ごとの対応の違いのまとめ。

## 1. 保留地とは？なぜローンが難しいのか

- **保留地（ほりゅうち）**: 土地区画整理事業の費用を捻出するために、事業主体（組合など）が売却する土地。
- **難しさの理由**: 事業が完了して「換地処分」が行われるまで、その土地の**不動産登記（表題登記・権利登記）ができない**ため。
- **銀行のリスク**: 通常、住宅ローンは土地に「抵当権」を設定して融資するが、登記がない保留地には抵当権が設定できない。

## 2. 金融機関ごとの対応状況（フラット35を含む）

| 金融機関タイプ | 対応状況 | 特徴・理由 |
| :--- | :--- | :--- |
| **ネット銀行** (住信SBIなど) | **原則不可** | 事務を定型化・効率化しているため、登記のない特殊な物件は審査対象外。 |
| **ARUHI（アルヒ）** | **対応可能** | 住宅ローン専門としてノウハウが豊富。フラット35（買取型）などで実績多数。 |
| **地元の地方銀行・信金** | **対応可能** | その地域の区画整理事業を把握しており、組合と連携して融資を行う。 |
| **ハウスメーカー提携ローン**| **対応可能** | HMが販売している保留地であれば、必ず対応する銀行を確保している。 |

## 3. フラット35における注意点

- **買取型 vs 保証型**:
    - **買取型**: 住宅金融支援機構の基準に準拠するため、保留地でも比較的利用しやすい。
    - **保証型**: 銀行独自の基準が加わるため、住信SBIネット銀行のように「保留地は不可」と明記されているケースが多い。
- **必要書類**: 土地区画整理組合が発行する「保留地証明書」や「権利譲渡通知」などの特殊な書類が必要になる。

## 4. 検討の進め方とアドバイス

1.  **土地の種別を早期に確認**: 検討中の土地が「保留地」や「仮換地」である場合、即座にネット銀行以外の選択肢（ARUHIや地銀）をリストアップする。
2.  **ハウスメーカーへの相談**: 「保留地でも通るローン」をHMに紹介してもらうのが最も確実。
3.  **ARUHIでの事前審査**: フラット35を希望する場合、ARUHIの店舗で「保留地だがどのプランなら可能か」を直接相談するのが早い。

## 5. メリット・デメリット（補足）

- **メリット**: 保留地は相場より少し安かったり、仲介手数料がかからない（組合直接販売の場合）などの金銭的メリットがある。
- **デメリット**: 住宅ローンの選択肢が狭まる、つなぎ融資の手続きが少し複雑になる場合がある。
