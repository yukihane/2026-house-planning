# 住宅ローンの検討プロセスと基礎知識

ハウスメーカー（HM）選びと並行して進めるべき住宅ローンの検討手順と、知っておくべき基本事項のまとめ。

## 1. 検討の全体スケジュール

住宅ローンの手続きは、家づくりのフェーズに合わせて以下のように進む。

1.  **HM選びの初期（今ここ）**:
    - 「いくらまで借りられるか」の把握。
    - ネット銀行などで**事前審査（仮審査）**を1〜2社通しておく。
2.  **HM決定・契約前後**:
    - 具体的な見積もりに基づき、本審査の準備。
    - 提携ローンと独自ローンの比較。
3.  **着工前後（土地先行取得の場合など）**:
    - 必要に応じて「つなぎ融資」や「分割融資」の手配。
4.  **建物完成・引き渡し**:
    - **本実行（融資実行）**。ここから返済が始まる。

## 2. 「事前審査（仮審査）」の重要性

- **目的**:
    - 自分の借入可能額を確定させ、予算オーバーを防ぐ。
    - HMに対して「支払い能力があること」を証明し、真剣な商談に進む。
- **特徴**:
    - ネット銀行なら数日〜1週間程度で結果が出る。
    - 複数の銀行に出しても信用情報に悪影響はない（短期間に多すぎなければOK）。
    - この段階ではHMが決まっていない「概算予算」でも審査可能。

## 3. 借入金額の決め方：借りられる額 vs 返せる額

- **借りられる額**: 年収、勤続年数、他社借入状況から銀行が判断する上限。
- **返せる額**:
    - 現在の家賃、今後の教育費、老後資金を考慮した現実的な返済額。
    - 一般的に**返済負担率（年収に対する年間返済額の割合）を25%以内**に抑えるのが安全とされる。

## 4. 金融機関選びの3つの軸

1.  **金利タイプ**:
    - **変動金利**: 低金利だが上昇リスクあり。
    - **固定金利（全期間・期間固定）**: 返済額が確定し安心だが、金利は高め。
2.  **団体信用生命保険（団信）**:
    - 万が一の際、ローン残高が0になる保険。
    - 最近は「がん診断で50%または100%保障」など、特約の充実度が銀行選びの決め手になることが多い。
3.  **諸経費**:
    - **事務手数料**: 借入額の2.2%（税込）が一般的。
    - **保証料**: 銀行に支払うのではなく保証会社に支払う。ネット銀行は保証料無料（その分手数料が必要）が多い。

## 5. 提携ローン vs 独自ローン

| 比較項目 | ハウスメーカー提携ローン | 独自ローン（ネット銀行など） |
| :--- | :--- | :--- |
| **金利** | 優遇される場合がある | 市場最安水準であることが多い |
| **手間** | 担当者が書類回収など代行してくれる | 全て自分で手配・郵送が必要 |
| **つなぎ融資**| スムーズに連携可能 | 対応していない銀行もあり注意 |
| **審査の通りやすさ** | 提携枠で通りやすい場合がある | 審査基準が比較的厳格 |

## 6. フラット35の「買取型」と「保証型」の比較

全期間固定金利の代表格であるフラット35には、主に以下の2つの仕組みがある。特にSBIグループなどのネット系金融機関では「保証型」が低金利で注目される。

| 比較項目 | フラット35（買取型） | フラット35（保証型） |
| :--- | :--- | :--- |
| **仕組みの要点** | 機構が銀行からローンを買い取る（一般的） | 銀行が貸し出し、機構が保険（保証）を引き受ける |
| **自己資金（頭金）** | 制限なし（融資率9割超も可） | **少なくとも10%以上が必要** |
| **金利の傾向** | 標準的。融資率（9割以下/超）で決まる | **買取型よりさらに低いことが多い** |
| **追加融資** | 「フラットパッケージローン」との組み合わせが可能 | 基本的に自己資金が必要な商品 |
| **審査の傾向** | 機構の基準（比較的柔軟） | 機構＋銀行独自の基準（比較的厳しい） |

- **買取型に向いている人**: 自己資金を抑えたい、または審査に不安がある場合。
- **保証型に向いている人**: 自己資金が1割以上あり、全期間固定で最安水準の金利を狙いたい場合。
